# Legends ZA Item Codes Changelog
Here's a list of updates and changes made to this repo, sorted by date. Please refer to the commits history for specific commit details.

### October 14th
#### New
- Created the repo!
- Uploaded item codes for most items.

---

### October 15th
#### New
- Added item code for the `Elevator Key` key item.
- Added codes to remove key items.
- Added a complete list of items present in Legends ZA as of patch 1.0.1, as well as their inventory hex values.
#### Updates
- Updated the README to credit the author of the code structure used by the item codes.
- Updated individual code files to credit the author of the code structure in use.
- Adjusted labelling on the "Remove Key Items" codes for improved clarity.
#### Fixes
- Misc. typo fixing.
- Corrected category value on Poke Ball item codes, resolving a known issue where Poke Balls that were cheated in did not appear in the throwing menu.
- Corrected category value on Berry, Key Item, Medicine, and Treasure item codes.
- Corrected inventory hex value of `Lumoise Galette` in item list.
- Added fossil items and hex values to item list, as they had been mistakenly omitted when the list was created.

### October 17th
#### New
- Added initial list of Pokemon encounter codes for use with the main encounter code found on gbatemp.

- Updated credits in item code files to credit the author of the new code sturcture in use.

- Corrected category value on TM and Mega Stone item codes.
